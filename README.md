# PlateSR-UNet

PlateSR-UNet is a deep learning project for enhancing low-resolution license plate images using a U-Net based super-resolution model.

This project was developed for portfolio and learning purposes.

---

## 🚀 Overview

Low-resolution license plate images can significantly reduce recognition performance.  
This project focuses on reconstructing higher-quality images from degraded inputs using a convolutional encoder–decoder (U-Net) architecture.

Main features:

- U-Net based super-resolution model
- End-to-end PyTorch training pipeline

---

## 🏗 Model Architecture

The model is based on the U-Net architecture with concatenation to preserve spatial details during upsampling.

Input: Low-resolution license plate image  
Output: Reconstructed high-resolution image  

---
